# Host Info

## hostInfoOK
1. Successfully read service index = 0 from delta.
2. Encode each element of t into m, then write to memory.

## hostInfoOOB
1. Successfully read service index = 0 from delta.
2. Encode each element of t into m, but encountered a permission error when writing m to memory.

## hostInfoNONE
1. Service index does not exist in delta.
